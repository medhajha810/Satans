# SatAns Website - Image Assets Guide

## Required Images

This document lists all the images you need to add to complete the website design.

### Main Images Folder (`images/`)

1. **hero-person.png**
   - Person sitting on red bean bag with laptop
   - Size: ~500px width
   - Background: transparent or light

2. **contact-person.png**
   - Person in pink top with laptop/phone
   - Size: ~400px width
   - Background: gradient (pink glow effect)

## Quick Setup Tips

### Option 1: Use Placeholder Images (for testing)
You can use placeholder services while you gather real images:
- https://placehold.co/500x500/E84855/white?text=Hero
- https://placehold.co/300x200/EF476F/white?text=Portfolio

### Option 2: Extract from Canva Design
1. Download your Canva design elements
2. Export images individually
3. Resize to recommended dimensions
4. Place in appropriate folders

### Option 3: Use Free Resources
- **Unsplash.com** - Free high-quality photos
- **Flaticon.com** - Free icons (SVG)
- Official brand websites - For brand logos
- **Pexels.com** - Free stock photos

## Image Optimization
Before adding images, optimize them:
- Use **TinyPNG.com** to compress PNG files
- Use **Squoosh.app** for JPG optimization
- Keep file sizes under 200KB when possible
- Use WebP format for better performance
